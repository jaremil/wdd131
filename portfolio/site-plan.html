Purpose

The purpose of this website will be a place link all of my personal coding projects in one place to show my ability to create, complex design, projessional and quality.

The audience of this website will be hiring managers and those representing companies at Job Fairs.

The website would have a filter to look up projects as well as a random select feature. The projects linked to the website will be stored in an Array.
